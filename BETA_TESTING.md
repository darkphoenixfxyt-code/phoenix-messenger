# Phoenix Messenger Private Beta Checklist

Version: v0.1.0-beta

Use this checklist for each private beta test pass with 3-5 trusted testers.

## Account

- [ ] Login works
- [ ] Username setup works
- [ ] Profile opens from settings
- [ ] Logout works

## Friends

- [ ] Search for a user
- [ ] Add friend
- [ ] Accept friend request
- [ ] Friend appears in friends list
- [ ] Start DM from friend

## Direct Messages

- [ ] Create a DM
- [ ] Send text message
- [ ] Edit own message
- [ ] Delete own message
- [ ] Reply to a message
- [ ] Add/remove reactions
- [ ] Unread badge appears for inactive DM
- [ ] Typing indicator appears
- [ ] Send image
- [ ] Send file/attachment

## Servers and Channels

- [ ] Create server
- [ ] Default channels appear
- [ ] Create invite
- [ ] Join invite from another account/browser
- [ ] Send channel message
- [ ] Send image in channel
- [ ] Delete own channel message
- [ ] Delete another user's channel message as owner/mod
- [ ] Kick a lower-role member as owner/admin
- [ ] Leave server as non-owner

## Voice and Calls

- [ ] Join voice channel
- [ ] Voice connection status changes to connected
- [ ] Mute/unmute
- [ ] Deafen/undeafen
- [ ] Leave voice channel
- [ ] DM voice call connects
- [ ] DM video call connects
- [ ] Call end clears call UI
- [ ] LiveKit token failure shows a toast
- [ ] Mic/camera denied shows a toast

## Mobile

- [ ] Open on phone browser
- [ ] Sidebar opens/closes
- [ ] Servers/channels are reachable
- [ ] DMs are reachable
- [ ] Input bar is not hidden by keyboard
- [ ] Tap targets feel usable

## Stability

- [ ] Refresh keeps chats
- [ ] Firestore permission denied shows a toast
- [ ] Image upload failure shows a toast
- [ ] Offline state shows in connection pill
- [ ] Reconnecting state shows in connection pill
- [ ] Netlify production URL loads
